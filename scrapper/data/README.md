# Data directory for the Ibtikar Chatbot

This directory contains:

## Subdirectories:
- `uploads/` - User uploaded documents
- `vector_store/` - Vector database files (ChromaDB/FAISS)
- `knowledge_base/` - Processed knowledge base documents
- `embeddings/` - Cached embedding files
- `models/` - Downloaded ML models

## Files:
- `sample_documents/` - Sample Arabic documents for testing
- `community_faq.json` - Community FAQ data
- `events.json` - Community events data

**Note:** This directory is excluded from version control via .gitignore
