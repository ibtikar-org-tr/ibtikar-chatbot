# Documentation

This directory contains comprehensive documentation for the Ibtikar Chatbot project.

## Structure:

### API Documentation
- `api/` - API endpoint documentation
- `schemas/` - Request/response schema documentation

### Architecture
- `architecture/` - System architecture diagrams and explanations
- `database/` - Database schema and ERD documentation
- `deployment/` - Deployment guides and configurations

### User Guides
- `user-guide/` - End-user documentation
- `admin-guide/` - Administrator documentation
- `developer-guide/` - Developer setup and contribution guide

### Technical Specifications
- `nlp/` - Arabic NLP processing documentation
- `rag/` - RAG pipeline documentation
- `security/` - Security implementation details

### Examples
- `examples/` - Code examples and usage samples
- `tutorials/` - Step-by-step tutorials

## Building Documentation

Documentation is built using MkDocs with Material theme:

```bash
pip install mkdocs mkdocs-material
mkdocs serve  # Development server
mkdocs build  # Production build
```
